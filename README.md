# hrax.app

GitHub-inspired landing page built with Next.js 14 (App Router) and Tailwind CSS.

The design draws from popular open-source web design projects on GitHub —
shadcn/ui's clean composition, Vercel/Linear's dark gradients, and
Aceternity/Magic UI's bento layouts.

## Sections

- Sticky glass navbar with brand chip
- Hero with gradient headline, animated dashboard preview, social proof
- Marquee logo cloud
- Bento feature grid (Components, Theming, CLI, Insights)
- Code preview side-by-side
- Masonry testimonials
- 3-tier pricing
- CTA with grid background
- Footer with wordmark and status indicator

## Getting started

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- TypeScript

All content is dummy/sample data.
